# 3D Solar System Simulation 🌌

This is a fully interactive, mobile-optimized 3D solar system built using Three.js.

## 🚀 Features
- Sun and 8 planets with real orbital mechanics
- Rings, atmosphere, realistic glow
- Star field + Nebula backdrop
- Speed control sliders
- Pause / Resume / Reset

## 🧩 Tech Stack
- HTML + CSS + JavaScript (ES6)
- Three.js r155
- Pure JS UI (no frameworks)

## 📦 To Run
Just open `index.html` in any modern browser (desktop or mobile).
